## 5. Creative Strategy
| Field | Value |
|-------|-------|
| **Hook strategy** | Shock Number — "$4,652 leaked" creates immediate information gap |
| **CTA** | "Try it free" — Level 3 (low friction) |
| **Framework** | PAS (Problem → Agitate → Solve) |
| **Justification** | Hidden costs = pain/fear topic. PAS amplifies the pain before offering relief, creating urgency to act. |
| **Primary visual concept** | Kinetic typography with counter animations |
| **Focal point strategy** | Each scene isolates one focal element via scale or color contrast |
| **Composition name** | `SubscriptionCreep` |
